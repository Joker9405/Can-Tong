# 使用与词源策略（MVP）

- **代码许可**：MIT（除 `/data` 与 `/private` 目录外）。
- **词库策略**：默认闭源，不公开完整数据集。`/data/seed.csv` 仅作演示展示；许可：**CC BY‑NC‑ND 4.0**（署名‑非商业‑禁止演绎）。
- **贡献协议**：对外收集的数据/范例描述，经贡献协议（CONTRIBUTION_CLA）授予项目方 **全球、可转授权、可商业** 的永久许可；公开面仅展示不含敏感信息，不公开原始全量。  
- **反抓取**：限频/禁止高频批量接口；严禁批量抓取、二次发布或商用再用。  
- **商业使用**：任何商业化使用必须获得项目方书面许可。
